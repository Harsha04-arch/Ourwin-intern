# T1

Very simple Hello World demo for CDN content generation based on requirements.

## What is in this project

- `index.html`: basic page structure for the demo.
- `styles.css`: minimal styling for the page.
- `app.js`: simple client-side behavior for the content generation flow.

## Simple instructions

1. Open `index.html` in your browser.
2. Review or update the requirement text in the page/script as needed.
3. Refresh the page to see the generated Hello World style content output.
